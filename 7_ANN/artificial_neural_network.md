# Building ANN with PyTorch

### ANN/MLP - Multi Layereed Perceptron


Dataset - Fashion MNIST
- imported from kaggle
- 28 x 28 images data of clothes

Architecture - 
1. Input Layer:
     - 784 Nodes
2. Hidden Layers
    - 1st hidden layer - 128 Neuron
    - 2nd hidden layer - 64 Neuron
3. Output Layer:
    - 10 Neuron
